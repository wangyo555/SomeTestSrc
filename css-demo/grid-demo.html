border-radius详解

正确的写法是border-radisu: 10px / 10px;

分别代表四个角的水平直径和垂直直径。

box-shadow详解

box-shadow: 5px 5px 5px 5px #ccc;

后面的参数分别代表：水平偏移、垂直偏移、模糊距离、阴影尺寸。其中阴影尺寸就是阴影外延出去总的长度；阴影的外延长度则是设定值的1/2。
